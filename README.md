# FabricAir Nordic Circular Polyester Research

Research platform mapping Nordic recycled polyester sources for FabricAir's circular ventilation textiles.

## Project Overview

**FabricAir** (Denmark) is a global leader in textile-based ventilation systems serving 120+ countries. This research maps **Finland, Sweden, and Norway** to identify sources of high-quality recycled polyester fiber.
