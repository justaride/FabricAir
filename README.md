# FabricAir Nordic Circular Polyester Research

Research platform mapping Nordic recycled polyester sources for FabricAir's circular ventilation textiles.

## Live Platform

**https://justaride.github.io/FabricAir/**

---

## Project Overview

**FabricAir** (Denmark) is a global leader in textile-based ventilation systems serving 120+ countries. This research maps **Finland, Sweden, and Norway** to identify sources of high-quality recycled polyester fiber.

## Key Finding

FabricAir can access **56,000-62,000 tonnes/year** of recycled polyester capacity by 2028-2029:

| Country | Partner | Capacity | Timeline |
|---------|---------|----------|----------|
| Finland | Rester | 6-12k t/yr | **Now** |
| Norway | NTG | 30k t/yr | ~2029 |
| Sweden | Rewin | 20k t/yr | Oct 2027 |

**Recommendation:** Start with Rester (Finland)—operational today.

## Material Requirements

| Parameter | Requirement |
|-----------|-------------|
| Material | 100% Polyester (PET) |
| Weight | ~300 g/m² |
| Color | White preferred |
| Source | Durable workwear, uniforms |
| **NOT suitable** | Polyester/cotton blends (65/35) |

---

## Critical Findings (Phase 2)

| Finding | Impact |
|---------|--------|
| **Lindström uses 65:35 blends** | 400 t/year INCOMPATIBLE |
| **Volume estimate revised** | 5-10k → 350-1,000 t/year |
| **LSJH Turku plant cancelled** | Rester is only Finnish recycler |
| **NTG handles poly-cotton** | Opens workwear access |
