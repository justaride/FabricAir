# Q2: Value Chain Stakeholders

> Who are the main stakeholders in production, use, and reuse of polyester textiles in Finland?

**Status:** Verified

---

## Summary

Finland's textile ecosystem includes 200+ companies with mandatory collection since January 2023 (2 years ahead of EU).

---

## Value Chain Overview

```
Production → Use → Collection → Sorting → Recycling → New Products
```
